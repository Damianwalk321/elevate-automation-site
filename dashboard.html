Architecture Note

One login, many modules

This dashboard should remain the single authenticated shell for the Elevate platform. Profile, compliance, AI, poster, billing, and referrals all sit under the same login and user identity.

Recommended structure: keep this page as the primary control center, then wire deeper modules like Profile & Setup into sub-pages or loaded panels without introducing a second auth flow.
